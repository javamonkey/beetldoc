设置chm文件名

文档项目下 xsl/chm 文件夹内的 chm-style.xsl 文件。

<xsl:param name="htmlhelp.chm">zoeeydoc.chm</xsl:param>

修改为生成 example.chm 则为：

<xsl:param name="htmlhelp.chm">example.chm</xsl:param>
